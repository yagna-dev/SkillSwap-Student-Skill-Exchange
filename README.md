# Student-Skill-Exchange-Platform
SkillSwap is a peer-to-peer student skill exchange platform where learners can teach, learn, and collaborate by exchanging technical skills within the CSE domain. It fosters mentorship, practical learning, and community-driven growth.
# SkillSwap

A peer-to-peer student skill exchange platform prototype designed for CSE students to share and learn technical skills collaboratively.

## Features
- Skill exchange concept
- User-friendly interface
- Peer collaboration model
- Green and white themed UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (if embedded)
